# Dino Adventure - Kids Dinosaur Games Collection

A collection of three educational and entertaining dinosaur-themed games designed for children.

## Games

1. **Dinosaur Runner Game** - An endless runner where kids control a dinosaur avoiding obstacles
2. **Dino Memory Match** - Match pairs of different dinosaur species to improve memory and concentration
3. **Dino Quiz Game** - An interactive educational quiz all about dinosaurs and prehistoric times

## Technology Stack

- **React Native with Expo** - Cross-platform development for iOS and Android
- **React Native Web** - Web platform support
- **TypeScript** - Type safety and better development experience

## Target Platforms

- iOS
- Android
- Web
